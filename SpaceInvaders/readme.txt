Интерактивная игра: "Space Invaders"
Запускается из program.py, для запуска требуется Pygame.
Управление:
x - стерльба
Стрелка влево - движение влево
Стрелка вправо - движение вправо
На экранах "проигрыша", "победы" и на стартовом экране требуется нажать любую клавишу для продолжения.
